Асинхронний скрипт для сортування файлів із вихідної папки по підпапках цільової папки на основі розширень файлів.

## Встановлення та запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

python3 sort_files.py <source> <output>
```

## Аргументи

| Аргумент | Опис |
|---|---|
| `source` | Шлях до вихідної папки з файлами |
| `output` | Шлях до цільової папки, де буде створено підпапки |

## Принцип роботи

- Скрипт рекурсивно обходить `source` разом з усіма підпапками.
- Для кожного файлу визначається розширення через `Path.suffix` (`.tar.gz` → підпапка `gz`, стабільна поведінка).
- Файли без розширення потрапляють у підпапку `no_extension`.
- Усі файли копіюються - оригінали залишаються на місці.
- Підпапки у `output` створюються автоматично.
- Копіювання файлів виконується паралельно через `asyncio.gather`.
- Помилки логуються через `logging` і не зупиняють обробку інших файлів.
